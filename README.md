# Ecstatic Aesthetics Website

## Site Description
Ecstatic Aesthetics is the personal portfolio and professional showcase of Bianca L Beltran, a passionate graphic designer specializing in web design, branding, and illustration. This website highlights Bianca's skills, past projects, and provides an easy way to get in touch. The site is designed to be accessible, responsive, and user-friendly across all devices.

The website features a consistent header, navigation, and footer across all pages. It includes images, media, and a contact form with proper validation. The Contact page now includes a Google Maps iframe showing Bianca’s office location and uses semantic form elements including `fieldset` and `legend`.

---

## Pages

1. Home (index.html)
   - Introduces Bianca L Beltran
   - Features a biography and profile image
   - Highlights skills and design philosophy

2. Portfolio (portfolio.html)
   - Showcases Bianca's past projects
   - Includes images of website designs, branding, and other creative work
   - Each project has descriptive alt text for accessibility

3. Contact (contact.html)
   - Provides a contact form for inquiries and collaborations
   - Form includes fields for Name, Email, Phone, Preferred Contact Method (fieldsetlegend), and Message
   - Validation with required attributes and `minlength` on the message
   - Embedded Google Maps iframe for office location
   - Fully responsive layout for mobile and desktop

4. About (About.html)
   - Displays a high-resolution image of Bianca's resume
   - Includes a clickable link to download the resume PDF
   - Designed to be responsive for mobile and desktop viewing

---

## Key Features
- Accessibility Single `h1` per page, semantic headings, alt text for all images, labels connected to inputs
- Responsive Design Flexible images, media queries, responsive iframes, and flexible layouts
- Consistent Navigation & Footer Same nav links and footer across all pages
- Interactive Contact Form Validated inputs, grouped options, required fields, and clear messaging
